Markdown# Análise de Satisfação de Restaurantes em Nova York

## Objetivo
Identificar quais tipos de culinária e bairros de Nova York têm os clientes mais satisfeitos, e descobrir padrões demográficos que influenciam as avaliações.

## Dataset
- 200 avaliações de restaurantes em diferentes bairros de NY
- Variáveis: culinária, localização, idade, orçamento, avaliações de comida/serviço, etc.

## Principais Descobertas
- **Melhor combinação**: Japonês no Upper East Side (aproximadamente 4.8/5.0)
- **Pior desempenho**: Italiano no Upper West Side (aproximadamente 2.0/5.0)
- Clientes acima de 50 anos são os que mais aprovam todos os restaurantes
- Orçamento alto está diretamente ligado a avaliações melhores

## Tecnologias
Python • Pandas • Seaborn • Matplotlib
